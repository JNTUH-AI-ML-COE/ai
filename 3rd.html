Course Description

This advanced course explores agentic AI: autonomous intelligent agents capable of perceiving, reasoning, planning, and acting to accomplish complex goals in the real world. Students will learn to build agents using LLM reasoning, agentic frameworks, tool use protocols, and multi-agent collaborationâ€”covering topics from agent memory and planning to security and system design.
Applications span coding assistants, web automation, robotics, scientific discovery, and more.

Learning Outcomes

  • Design full agent architectures: memory, tool APIs, planning/logic, collaboration.
  • Apply frameworks for agent reasoning (Chain-of-Thought, ReAct) and tool use.
  • Integrate LLMs with tools, APIs, databases, and feedback loops.
  • Evaluate and debug agent performance and collaboration (multi-agent, human-in-the-loop).
  • Implement ethics and security: prevent unsafe behaviors and secure tool use.
  • Deploy applied agent systems, culminating with a capstone project.

Prerequisites

  • Graduate-level background in machine learning/AI (CS188/189 or equivalent)
  • Strong Python programming skills
  • Exposure to NLP or reinforcement learning is beneficial
  • Active interest in research papers and engineering complex systems

Course Format & Assessment

  • Weekly seminars: lectures + discussion of agentic AI research and demos
  • Bi-weekly programming assignments & mini-projects (hands-on agent building and security)
  • Midterm quiz on core concepts and research
  • Capstone Project: Design, build, and demonstrate a fully functional agent
  • Grading: 50% project, 30% assignments, 10% paper presentation, 10% participation

Weekly Syllabus

Week 1: Introduction to AI Agents

  • Defining agents vs. chatbots; multi-step reasoning, persistent state, goal pursuit
  • Evolution of agent systems; examples across domains
  • Reading: Survey on LLM Agents
Mini-project: Run and analyze an open-source agent (AutoGPT or AgentGPT) on a real-world goal.

Week 2: LLMs as Reasoning Engines & Prompting

  • Using LLMs for step-by-step reasoning (Chain-of-Thought, CoT)
  • ReAct framework: interleaving reasoning and tool use; prompt engineering for agents
  • Self-feedback and reliability improvements
  • Readings: Stanford CS329A Agent Architectures, ReAct: Reason+Act
Hands-on: Write and test prompts for tool-using agents, compare with/without CoT prompting.

Week 3: Agent Architecture and Frameworks

  • Agent system components: planning, memory, orchestration, tools
  • Classical symbolic vs. modern data-driven and LLM-based agents
  • Survey of top agent frameworks: LangChain, Crew, AutoGen
  • Reading: Agent Engineering Syllabus PDF
Diagram architecture of a modern agent; mini-project using LangChain or similar.

Week 4: Tool Use and Agent Interoperability Protocols

  • Plugin/function integration (APIs, databases, web automation)
  • Introduction to Model Context Protocol (MCP), Agent-to-Agent (A2A) comms, AG-UI standards [MCP, A2A, AG-UI]
Lab: Implement MCP client to enable external tool use for your agent.

Week 5: Multi-Agent Systems and Collaboration

  • MAS: cooperation, competition, distributed task allocation
  • Multi-agent protocols and prompt optimization (MASS, A2A)
  • Reading: Multi-Agent Design, Prompt/Topology Optimization
Assignment: Script and analyze multi-agent interactions and team performance.

Week 6: Memory and Knowledge Management

  • Persistent agent memory: vector stores, MemGPT, retrieval-based architectures
  • Knowledge graphs, episodic/semantic memory, recall and up-to-date info
  • Reading: MemGPT benchmarks
Augment agent with conversation and tool memory; empirical analysis of context improvements.

Week 7: Planning and Decision-Making

  • Classical AI planning, LLM agent planning, decision trees, Tree of Thoughts
  • Integrating lookahead and goal decomposition in agents
  • Reading: Language Agent Tree Search, Oxford Self-Programming Agents
Assignment: Implement stepwise plan execution for a multi-step task using LLMs or planning libraries.

Week 8: Learning, Adaptation, and Self-Improvement

  • Reinforcement learning, self-reflection, automated self-correction (Constitutional AI)
  • Agent evolution, goal generation, continual improvement; case studies in code, science agents
  • Reading: Self-Improving Agents CS329A
Lab: Build feedback/self-correction loop for agents and analyze learning curves.

Week 9: Evaluation and Benchmarking

  • Metrics and benchmarks for agent abilities, open-ended tasks, robustness, safety
  • Tools: OpenAI Evals, BrowseBench, SWE-Bench, HELM, AgentBench
  • Reading: Survey on Evaluation of LLM Agents
Assignment: Create and run agent benchmarks, compare agents on web, code, or dialogue tasks.

Week 10: Agent Safety and Security

  • Prompt injection attacks, design patterns for secure agents, compliance and alignment
  • Sandboxing, threat modeling, role separation, secure tool access
  • Reading: Securing LLM Agents - Design Patterns
Lab: Simulate attacks and defenses in agent prompts; implement mitigation strategies.

Week 11: Agents in Software Engineering and DevOps

  • Copilot systems, autonomous code agents, SWE-Agent for code generation/maintenance
  • DevOps: agents for software delivery, bug fixing, infrastructure management
  • Reading: SWE-Agent research
Project checkpoint: Review agent designs for code/tooling; demo and feedback sessions.

Week 12: Agents in the Physical World and Scientific Discovery

  • Robotics: grounding language in actions, integrating LLM agents with IoT
  • Scientific discovery: hypothesis generation, virtual experimentation, agent-based modeling
  • Open problems: reliability, domain knowledge, ethical considerations
Capstone demos: Present applied agent projects; discuss future challenges.
