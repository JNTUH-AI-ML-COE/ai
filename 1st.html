AI-Powered Modern Software Development

Comprehensive Curriculum & Course Details

About this course:
This curriculum provides a deep-dive into how AI and large language models (LLMs) are transforming software engineering. Each module below contains clear learning objectives, hands-on topics, and the latest industry references to prepare you for the future of development.
Week 1: Introduction to Coding LLMs and AI Development
  • Understand large language models (LLMs) and their code generation
  • Learn effective prompt engineering for software tasks
  • Explore why LLMs sparked a new paradigm (“vibe coding”) and how to harness them responsibly
Topics & Subtopics
  • Foundations of LLMs: Overview, transformer-based LLMs, and their training on code
  • Code-Specific LLMs: GPT-4.1, Claude 4, Code Llama-2 (capabilities as of 2025)
  • Prompt Engineering Basics: Clear prompts, specifying language, step-by-step, function signatures
  • Evolving Best Practices: Formatting, reasoning scaffolds, role prompting, model-specific styles
  • Prompting for Reliability: Secure prompting, input validation, constraints
  • Impact on Workflow: AI coding assistants (e.g. GitHub Copilot), productivity and acceptance, remaining limitations
  • “Vibe Coding” Concept: Natural language prototyping for software, democratization, security omissions
References:
  • Who's Coding Now? AI and the Future of Software Development (a16z Podcast)
  • The Ultimate Guide to Prompt Engineering in 2025 | Lakera
Week 2: The Anatomy of Coding Agents
  • Break down an AI coding agent into core components
  • Build an LLM-based coding agent (planner, tools, memory, MCP)
  • Understand planning, control, tool APIs, Model Context Protocol
Topics & Subtopics
  • Planning & Autonomy: Workflow planning, improved planning algorithms, managing errors
  • Tool Use & Function Calling: Standardized APIs for external tools, code execution, documentation queries
  • Model Context Protocol (MCP): Open standard for connecting tools, secure integration, industry adoption
References:
  • Anatomy of an AI Agent (IBM Blog)
  • Introducing the Model Context Protocol | Anthropic
Week 3: The AI IDE (Intelligent Development Environment)
  • Explore AI-powered IDEs, context management with embeddings
  • Practice AI pair programming and code understanding
  • Integrate AI with code review, doc generation, and navigation tools
Topics & Subtopics
  • Context Management: Embeddings for code retrieval, workspace context
  • Large Codebases: Summarization, long-context models, retrieval vs. full context
  • AI-Based Code Understanding: Bug finding, code explanations, multi-agent debugging
  • AI IDE Integrations: VS Code, Copilot agent mode, Open Source VS Subscription tools
  • Best Practices: Structured queries, manual validation, discipline in AI-driven development
References:
  • Introducing GitHub Copilot Agent Mode (VS Code Blog)
  • GitHub Introduces New Embedding Model | InfoQ
Week 4: Coding Agent Patterns - Autonomy and Collaboration
  • Evaluate agent autonomy, human-in-the-loop and collaboration designs
  • Apply tool orchestration, co-reviewer, pair programming patterns
  • Develop trust and audit trails in agentic workflows
Topics & Subtopics
  • Human/AI Hybrid Workflows: Autonomy spectrum, oversight, best practices
  • Patterns: Handoff, tool orchestrator, pair programming, co-reviewer with industry examples
  • Audit, Logging, Gradual Autonomy: Best practices for safe agent deployment
References:
  • Agent vs Human-in-the-Loop in 2025: The Definitive Comparison Guide | Skywork

  • Automate scripts and troubleshooting with AI
Topics & Subtopics

  • Sandboxed Environments: Containers, safe script execution
  • Terminal Automation: Natural language to script, looped workflows, intelligent CLI UI
